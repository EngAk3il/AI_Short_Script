## 2. Relationship with viewer (HOW he thinks about “you”)

- Viewer = **victim** of the system (“tum logon ko”, “tumhe”).
- He = **insider** who already figured it out (“main tumhe batata hoon”, “poora system samjho”).
- Tone: **accusatory toward companies**, protective toward audience — never insults the viewer.
- Mix **tum** (~60%) and **aap log** (~40%) — both OK in one script.

---

## 3. Mental model when picking a topic

Ask before writing:

1. **Who is the villain?** (company, founder, policy — name them in line 1)
2. **What did they do TO the viewer?** (price, IPO dump, illusion, natak)
3. **What’s the one number that makes it obscene?** (₹ must be exact, never “crores” vague)
4. **What’s the mechanism?** (how the scam/illusion works — step by step)
5. **Mic-drop line?** (no soft landing — end on verdict, often no CTA)

---

## 4. Emotional arc (every script)

```
OUTRAGE (0–5s) → “prove it” tension (5–20s) → evidence stack (20–45s) → villain/motive reveal (45–60s) → brutal close (60–75s)
```

Never start warm. Never end with “thanks for watching.”

| Connectors | **“ke andar”** constantly; **“bhai”**; **“par usse pehle mudda samjho”** |

## 6. Body structure (copy this flow)

1. **Brutal verdict** — company/person failed / stole / natak (hook)
2. **“Main tumhe numbers dikhata hoon”** or **“poora poora system samjho”**
3. Stack **3–5 proofs** — each worse than the last
4. Compare to expose absurdity (“YouTube ₹200 le leta hai, ye ₹600 CPM…”)
5. **Sarcastic close** — optional rare CTA: “detail video aa rahi hai follow karo”
