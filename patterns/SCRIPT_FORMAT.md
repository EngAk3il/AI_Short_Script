# Script Format Rules — Standard Format for ALL Generated Scripts

> This document defines the EXACT format every generated script must follow.
> Analyzed from existing scripts: hellooipsita (4), SciCoLens (1), CleoAbram (1).
> NEVER deviate from this format.

> [!CAUTION] **ALL HINDI SCRIPTS MUST USE ROMANIZED HINDI (Roman/Latin letters)**
> Write "Kya aapne kabhi socha hai" — NOT "क्या आपने कभी सोचा है"
> This applies to ALL creators. NEVER use Devanagari script.

---

## Mandatory Structure

Every script `.md` file MUST contain these sections in this exact order:

### 1. Title Line
```
# [Catchy Title with Topic] [1-2 Emoji]
```
- Use the creator's language for the title (Hindi/Hinglish for Hindi creators, English for English creators)
- 1-2 relevant emoji at the end

### 2. Metadata Block
```
**Style DNA:** [CreatorName] ([Niche Description])
**Language:** [Hindi / Hinglish (Hindi + English) / English]
**Topic:** [Specific Topic / Category]
**Creator Angle:** [Identify the HIDDEN STORY, psychology, or specific mechanism. DO NOT write a news summary.]
**Status:** AI Generated (DNA v2 Adherence)
```

### 3. Script Table (THE CORE)
```
| Timestamp | Hindi Script | Pacing (WPS) | Context/Visuals |
|-----------|--------------|--------------|-----------------|
| [00:00] | [Script line in creator's language] | X.X | [Visual/camera direction] |
| [00:08] | [Next line] | X.X | [Visual] |
...
```

**Column Rules:**
- **Timestamp:** `[MM:SS]` format. Each row is ~6-10 seconds.
- **Hindi Script:** Column is ALWAYS called "Hindi Script" for Hindi/Hinglish creators. The actual text is in the creator's language/mix. For English creators, use "English Script".
- **Pacing (WPS):** Words per second, matching the creator's DNA WPS range.
- **Context/Visuals:** Camera directions, B-roll descriptions, text overlays, visual transitions.

### 4. Horizontal Rule + Fact-Check Sources
```
---
### Fact-Check Sources:
- [Source Name (Publication)](URL)
- [Source Name (Publication)](URL)
```

### 5. DNA Adherence Checklist
```
### DNA v2 Adherence Checklist:
- ✅ **[Pattern Name]:** [How it was used in this script]
- ✅ **[Pattern Name]:** [How it was used]
...
```

## Total Script Duration
- Target: **55-75 seconds** (YouTube Shorts sweet spot)
- Rows: **7-10 timestamp rows** typically
- Each row: **~6-10 seconds** of spoken content
