# 🚀 Deep Creator Learning: Prompt Library

Use these prompt templates to get the highest quality, high-retention scripts from the system. These are categorized by your "DNA" layers: **Creator Voice** (Tone) and **Strategy Expert** (Virality).

---

## 1. Core Generation (Creator DNA)
*Use these to generate a base script that sounds exactly like a specific creator.*

> **Prompt:** "Create a script for **[Creator Name]** about **[Specific Topic/Angle]**. Ensure it follows their DNA profile for tone, pacing, and vocabulary."

**Example:**
> "Create a script for **aevytv** about the 'Underground Mafia of Indian Click-Farms'. Use their relatable investigative documentary style."

---

## 2. Strategy Enrichment (Single Expert)
*Use these to add a layer of 'virality logic' to an existing or new script.*

> **Prompt:** "Enrich this script using **[Strategy Expert]**'s frameworks. Specifically, apply their **[Specific Formula]**."

**Example:**
> "Enrich this script using **Ethan Shaw's** 'Value-tainment' formula. Ensure the Hook leads immediately to Social Proof before delivering the Direct Value."

---

## 3. Triple-Enriched Mode (Advanced Hybrid)
*This is the 'Master Mode'. It blends one creator's voice with multiple strategy layers.*

> **Prompt:** "Generate a script in the voice of **[Creator]**, but structure the hook using **[Expert A]**'s formula and the body using **[Expert B]**'s retention switches."

**Example:**
> "Write a script in the voice of **aevytv**. Use **Alex Hormozi's** 'Price Anchoring' for the lead, but apply **TubeBuddy's** 'Perfect Loop' engineering for the closer."

---

## 4. Self-Correction & Audit
*Use these to ensure the system hasn't 'hallucinated' or become too robotic/generic.*

> **Prompt:** "Perform a **Creator Angle Audit** on this script. Compare it line-by-line against **[Creator]**'s DNA. Does it sound like them, or is it too generic? Fix any robotic sentences."

**Example:**
> "Perform a Creator Angle Audit for **Shivanshu.Agrawal**. Is the humor too forced? Are the transitions smooth? Rewrite any line that sounds like a generic AI."

---

## 6. V3 Master Synthesis (The Anti-Static Mode)
*This is the most advanced mode. It combines Creator DNA, Multiple Strategies, and a specific Narrative Arc.*

> **Prompt:** "Generate a V3 script for **[Creator]** about **[Topic]**. Use the **[Narrative Skeleton]** skeleton. Apply **[Expert A]** for the lead and **[Expert B]** for the body. Use **Dynamic Pacing (WPS Maps)** and **Strategy-Driven Visuals**."

**Example:**
> "Generate a V3 script for **hellooipsita** about 'Dopamine Hacks'. Use the **Mystery Loop** skeleton. Apply **Alex Hormozi**'s Price Anchor for the lead and **Matt Gray**'s System Logic for the body. Ensure **Dynamic Pacing** (slow for the mechanism, fast for the hook)."

---

## 7. Strategy Diagnostic
*Use these to check if a script is actually designed for retention.*

> **Prompt:** "Audit this script for **Retention Switches**. Does it have a 'Pattern Interrupt' every 10 seconds following **Ethan Shaw's** rules for momentum?"

---

### 💡 Pro-Tip for Best Results:
When asking for a script, always provide a **"Creator Angle"**.
*   **Bad:** "Write about Zomato."
*   **Good:** "Write about Zomato, specifically focusing on the psychology of why their notifications are so addictive (AevyTV style)."
